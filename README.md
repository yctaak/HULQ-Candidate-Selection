# $\color{#00D166}{\textbf{HULQ}}$ Data Pipeline

This repository contains core scripts for the $\color{#00D166}{\textbf{HULQ}}$ (High-$z$ Universe probed via Lensing by QSOs) project. 

### Contents
- `01_fetch_euclid.py`: A pipeline to cross-match SDSS/DESI catalogs with Euclid Q1 coverage, download multi-filter FITS cutouts, and generate RGB thumbnails.
